# Ostrich Smart Hub 🦢 [🚧 Work In Progress]

**Ostrich Smart Hub** adalah sistem manajemen tiket dan monitoring operasional digital yang dirancang khusus untuk **Ostrich Mini Zoo**, Subang. Sistem ini bertujuan untuk mendigitalisasi proses reservasi, ticketing, serta pemantauan satwa secara real-time.

---

## 🛠️ Tech Stack & Tools
*   **Framework:** Laravel (PHP)
*   **Frontend:** Blade Templating
*   **Database:** MySQL
*   **Design Tools:** Figma (Wireframing), Canva (Documentation)

## 🚀 Fitur yang Sedang Dikembangkan
*   **Digital Ticketing System:** Sistem pemesanan/booking tiket masuk kebun binatang secara online.
*   **Smart Monitoring Hub:** Dashboard pemantauan status hewan untuk manajemen satwa.
*   **EER Based Database:** Struktur database yang teroptimasi untuk kecepatan transaksi.
*   **Automated Reporting:** Laporan harian jumlah pengunjung otomatis.

## ⚙️ Cara Instalasi (Local Development)
1.  Clone repository ini.
2.  Jalankan `composer install` & `npm install`.
3.  Salin `.env.example` menjadi `.env` dan sesuaikan konfigurasi database.
4.  Jalankan `php artisan key:generate`.
5.  Jalankan `php artisan migrate`.
6.  Akses melalui `php artisan serve`.
